# 🛡️ CareNova
Personal Safety & Health Helpline for Hackathon 2026

### Features
- Multi-user login portal
- Personal health tracker: BP, heart rate, blood sugar per user
- One-tap emergency calling: Police 100, Ambulance 108, Fire 101
- AI Safety Assistant chatbot
- SOS alert with timestamp log
- Emergency contacts with click-to-call
